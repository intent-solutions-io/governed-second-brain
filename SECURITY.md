# Security Policy

This umbrella repo contains documentation only. Report security issues in the **flagship repos**
where the code lives:

- [bobs-big-brain-compiler](https://github.com/jeremylongshore/bobs-big-brain-compiler/security) (the compile engine)
- [bobs-big-brain-registrar](https://github.com/jeremylongshore/bobs-big-brain-registrar/security) (the govern engine)

For anything not covered by a repo-specific channel, email **security@intentsolutions.io**.
Please do not open public issues for vulnerabilities — disclose privately first.
